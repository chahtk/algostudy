## Round Function
### round 함수 사용시 주의 사항
```python
round(0.5) # 0
round(-0.5) # 0
round(1.5) # 2
round(2.225, 2) # 2.23
round(4.225, 2) # 4.22
```

### 원인
우리가 아는 반올림은 마지막 자릿수가 5 이상일 때 올림을 하지만, python의 내장 함수 `round()`는 마지막 자릿수가 **5**일 때, 짝수에 가까운 수로 올리거나 내린다.

### 소수(float)일 때
마지막 두 예시처럼 소수일 때, 부동소수 문제로 인하여 종잡을 수 없는 결과가 나올 수 있다. 확인해보고 사용할 것!
